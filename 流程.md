## 请求解析详细流程

```mermaid
flowchart TD
    Start[收到原始请求数据] --> CheckDBSize{DBSize > 0?}
    
    CheckDBSize -->|是| MergeData[合并数据块到BodyBuf]
    MergeData --> UpdateDBSize[DBSize -= 接收数据大小]
    UpdateDBSize --> CheckComplete{DBSize == 0?}
    CheckComplete -->|否| WaitMore[等待更多数据]
    CheckComplete -->|是| ParseSuccess[解析完成]
    
    CheckDBSize -->|否| ConvertToString[转换为UTF-8字符串]
    ConvertToString --> FindLineEnd[查找第一行结束位置]
    FindLineEnd --> FindHeadersEnd[查找请求头结束位置]
    
    FindHeadersEnd --> ValidateFormat{格式正确?}
    ValidateFormat -->|否| Return400[返回400错误]
    ValidateFormat -->|是| CheckProtocol{协议标识正确?}
    
    CheckProtocol -->|否| Return400
    CheckProtocol -->|是| ParseLine[解析请求行]
    
    ParseLine --> ExtractMethod[提取请求方法]
    ExtractMethod --> ExtractUrl[提取URL]
    ExtractUrl --> ExtractProtocol[提取协议版本]
    ExtractProtocol --> CheckQueryParams{有查询参数?}
    
    CheckQueryParams -->|是| ParseQueryParams[解析GET参数]
    ParseQueryParams --> ValidateParams{参数成对?}
    ValidateParams -->|否| Return400
    ValidateParams -->|是| URLDecode[URL解码]
    CheckQueryParams -->|否| URLDecode
    
    URLDecode --> ParseHeaders[解析请求头]
    ParseHeaders --> LoopHeaders[逐行解析]
    LoopHeaders --> FindColon{找到冒号?}
    FindColon -->|否| Return400
    FindColon -->|是| ExtractKeyValue[提取键值对]
    ExtractKeyValue --> ValidateHeaders{键值成对?}
    ValidateHeaders -->|否| Return400
    ValidateHeaders -->|是| StoreHeaders[存储到Headers Map]
    
    StoreHeaders --> ExtractBody[提取请求体]
    ExtractBody --> CheckPostChunked{POST且分块?}
    
    CheckPostChunked -->|是| CheckContentLength{Content-Length > body.size?}
    CheckContentLength -->|是| CacheRemaining[缓存剩余数据到DBSize]
    CheckContentLength -->|否| CompleteParse[完成解析]
    CheckPostChunked -->|否| CompleteParse
    
    CacheRemaining --> WaitNextBlock[等待下一块数据]
    CompleteParse --> ParseSuccess
```

## 主处理流程 (Main)

```mermaid
sequenceDiagram
    participant Client as 客户端
    participant Server as HttpServer
    participant Req as Request
    participant Res as Response
    participant Route as 路由函数
    
    Client->>Server: 建立连接
    Server->>Server: onACCEPT 触发
    Server->>Client: 创建client对象
    Client->>Server: 发送HTTP请求
    Server->>Req: Parse(原始数据)
    
    alt 数据不完整
        Req->>Req: 缓存数据块
        Req->>Server: 返回空(等待更多数据)
    else 解析失败
        Req->>Server: 返回错误码(400)
        Server->>Res: SetErrorRes(错误码)
        Server->>Server: SendResponse
        Server->>Client: 发送错误响应
    else 解析成功
        Req->>Server: 返回0
        Server->>Res: __New() 重置响应
        Server->>Server: PreHandleReq钩子
        
        alt 钩子拒绝
            Server->>Client: 关闭连接
        else 继续处理
            Server->>Server: HandleRequest
            
            alt API路由匹配
                Server->>Route: 执行路由函数(Req, Res)
                Route->>Res: 设置响应内容
            else Web请求
                Server->>Server: HandleWebRequest
                Server->>Res: SetBodyFile(文件路径)
            else 未找到
                Server->>Res: SetErrorRes(404)
            end
            
            Server->>Server: SendResponse
            Server->>Res: DefResLine/DefResHeader/DefResBody
            Server->>Server: PreSendRes钩子
            
            alt 钩子拒绝
                Server->>Client: 关闭连接
            else 发送响应
                alt Body是Buffer
                    Server->>Client: SendText(响应头)
                    Server->>Client: Send(二进制Body)
                else Body是文本
                    Server->>Client: SendText(完整响应)
                end
                
                alt 调试模式
                    Server->>Server: DeBug输出
                end
                
                alt Connection: close
                    Server->>Client: 关闭连接
                    Server->>Req: __New() 重置
                    Server->>Res: __New() 重置
                else keep-alive
                    Server->>Client: 保持连接
                    Note over Server,Client: 等待下一个请求
                end
            end
        end
    end
```

### Hook 使用示例

```autohotkey
; IP访问控制
server.onFunc["isIPAllow"] := (ip) => {
    return ip = "127.0.0.1"
}

; 请求预处理
server.onFunc["PreHandleReq"] := (req, res) => {
    ; 身份验证、限流等
    if not Authenticate(req.Headers["Authorization"]) {
        return false  ; 中断处理
    }
    return true  ; 继续处理
}

; 响应后处理
server.onFunc["PreSendRes"] := (req, res) => {
    ; 添加通用Header、日志记录等
    res.Headers["X-Powered-By"] := "AHK-HTTP"
    Log.Info("Response sent for " req.Url)
    return true
}
```

## 注意事项

⚠️ **重要提醒**:
- 路由函数必须至少接受2个参数(Request和Response)
- 响应只在SendResponse时最终构建
- Body为Buffer类型时,响应头和体分开发送
- HEAD方法会自动清空Body
- TRACE方法会回显完整请求
- 调试模式下(/debug或TRACE)会输出详细信息
- 日志自动保存到 `logs/YYYY-MM-DD.log`
